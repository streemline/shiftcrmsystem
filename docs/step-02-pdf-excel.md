## Что сделано

### `src/utils/pdfReports.js` — новый модуль профессиональной генерации PDF

#### `generateEmployeePDF(employee, records, yearMonth, companyName)`
Генерирует персональный PDF-табель для сотрудника. Содержит:
- **Шапка** — логотип WorkTime (векторный, jsPDF), название компании, тип отчёта, период
- **Карточка сотрудника** — ФИО, email, ставка (Kč/ч)
- **Детализация записей** — таблица по дням: дата, тип дня, объект, время, часы, заработок; чередующиеся строки (zebra), перенос на новую страницу
- **Итоговый блок** — рабочих дней, часов, к выплате (Kč), больничных/отпуск
- **Работа по объектам** — мини-карточки по каждому объекту (дней + часов)
- **Поле подписи** — сотрудник + руководитель + место печати (М.П.)
- **Футер** — номер страницы / всего страниц, дата генерации

#### `generateSummaryPDF(employees, allRecords, yearMonth, companyName)`
Сводный PDF по всем сотрудникам за месяц — таблица: ФИО, рабочих дней, часов, к выплате; итоговая строка.

### `src/utils/exportExcel.js` — обновлённый Excel-экспорт

| Функция | Описание |
|---------|----------|
| `exportWorkRecordsToExcel(records, filename)` | Записи рабочего времени со всеми полями + итоговая строка |
| `exportMaterialsToExcel(materials, filename)` | Материалы с количеством, ценой, итогом |
| `exportSummaryToExcel(employees, records, yearMonth)` | Сводная таблица по сотрудникам |
| `exportToCSV` | Сохранён для обратной совместимости |

Все файлы: UTF-8 BOM + CRLF — корректно открываются в Excel и LibreOffice с кириллицей.
